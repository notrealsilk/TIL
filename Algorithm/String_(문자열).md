# 문자열 비교

## 문자열 비교 연산


```python
# == : 요소가 같은지 확인
# is :  메모리 참조가 같은지 확인

s1 = 'abc'
s2 = 'abc'
s3 = s1[:2]+'c'

print(s1 == s2) # True
print(s1 == s3) # True
print(s1 is s3) # False
 
```
