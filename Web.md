💡 html 단축키
- 설치 후 실행 단축키 : `alt` + `B`
- vs 코드에서 `!`+ `tab`을 하면 기본 뼈대가 나옴

- `ctrl` + `alt` 하면 커서 여러 개
- `ctrl` `l` 하면 한 줄 선택
- `alt` 하고 여러 개 줄 선택, 방향키하면 한 번에 옮기기 가능
- `alt` `shift` 하면 여러 개 줄 복사
